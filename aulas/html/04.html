Strings

```python texto = "Aprendendo Python" texto = 'Aprendendo Python' texto = ''' Aprendendo Python ''' ```

String (escape)

Se quiser incluir o símbolo de aspas (simples ou duplas) em uma string, use a expressão \“ ou \’ . Ou use “ e ‘ aninhados:
```python bairro= "Ponta D'areia" bairro= 'Ponta D\'areia' ```

Acessando Caracteres em uma String

As strings são indexadas, iniciando por 0

```python str1[0] # primeiro caracter: 'P' str1[5] # ultimo caracter: 'n' str1[6] # ERRO!! ```

String

Índices também podem ser negativos, de trás para frente:

```python str1[-1] # ultimo caracter: 'n' str1[-6] # primeiro caracter: 'P' str1[-7] # ERRO!! ```

Acessando Parte de uma String (Slicing)


String

Acesso a partes da string (slicing)

```python str1[0:2] # posição 0 a 1: 'Py’ str1[:2] # posição inicial até a 1:'Py’ str1[2:6] # posição 2 a 5: 'thon’ str1[2:] # carac. da posição 2 até o final: 'thon’ str[:-2] # exclui os dois últimos caracteres: 'Pyth' ```

Manipulação de String

upper() - converte o conteúdo da string para letras maiúsculas:
```python >>> s = “Exercicio” >>> s.upper() EXERCICIO ```

Manipulação de String

lower() - converte o conteúdo da string para letras minúsculas:
```python >>> s = “BICT/UFMA” >>> s.lower() bict/ufma ```

Manipulação de String

Operador '+': Concatena duas ou mais strings.
```python >>> s1 = "Python" >>> s2 = "3" >>> s1 + " " + s2 'Python 3' ```

Manipulação de String

Operador '*': Faz cópias repetidas da string.
```python >>> s = "UFMA" >>> s*3 "UFMAUFMAUFMA" >>> (s+ ‘ ”)*3 "UFMA UFMA UFMA" ```

Manipulação de String

len(): quantidade de caracteres da string.
```python >>> s = "BICT/UFMA" >>> len(s) 9 ```

Manipulação de String

find(): retorna o índice da 1ª ocorrência de uma string dentro da outra
```python >>> nome='José da Silva' >>> nome.find('s') 2 >>> nome.find('da') 5 ```

Manipulação de String

Manipulação de String

in : Percorre uma string (iteração).
```python s = 'algoritmos' for x in s: print(x) ```

Manipulação de String

in : verifica se uma string é substring de outra.
```python >>> nome='José da Silva' >>>'da' in nome True >>> 'de' in nome False ```

Manipulação de String

split() : retorna os caracteres antes da primeira ocorrência de um caractere separador em uma string de entrada.
```python >>> str1 = 'Fulano de Tal' >>> str1.split() # ['Fulano', 'de', 'Tal'] >>> str1.split(' ') # ['Fulano', 'de', 'Tal'] >>> str1.split(',') # ['Fulano de Tal'] >>>'1,2,3'.split(',') # ['1', '2', '3'] ```
Obs: O caractere separador default é o espaço em branco.

Manipulação de String

replace() : realiza uma operação padrão de localizar e substituir. substitui strings para criar uma nova string.
```python S1 = 'um aluno, dois alunos, tres alunos.' S2 = S1.replace('aluno','estudante') print(S1) print(S2) ```

Manipulação de String

replace() : realiza uma operação padrão de localizar e substituir. substitui strings para criar uma nova string.
```python str1 = '''Na minha família há vários médicos. Meu avô, meu pai, minha mãe e eu somos médicos.''' str2 = 'médicos' str3 = 'engenheiros' str1.replace(str2,str3) ```

Manipulação de String

ITERAÇÃO : Percorre cada caractere uma string.
```python S = 'algoritmos' for x in S: print(x) ```
