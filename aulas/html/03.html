### Dados e Tipos de Dado > Um dado é uma informação que um algoritmo recebe ou manipula; Exemplos de dados são: - Nomes; - datas; - valores (preços, notas, etc.); - e condições (verdadeiro e falso).

Dados e Tipos de Dado

Todo dado é de um certo tipo que define sua natureza (p. ex., um nome é diferente de um valor numérico), identificando seu uso, e define as operações que podem ser realizadas com o dado;
Por exemplo: podemos somar dois valores numéricos, mas NÃO podemos somar um número e uma frase.

Dados e Tipos de Dado

Tipos de dados atômicos:

  • int (ex: 1; 36; 2017)
  • float (ex: 3,1416)
  • bool (ex: True; False)

Tipos de dados de coleção:

  • Listas
  • Tuplas
  • String
  • Dicionários
### ARRAY > Array é um termo genérico para sequência. Em Python temos 4 estruturas com comportamento de arrays ##### Lista, Tupla, Conjuntos (sets) e String - Todas suportam indexição
## TUPLAS
### TUPLAS - Uma tupla é uma coleção ordenada de zero ou mais valores, de um mesmo tipo ou não. Ela serve para agrupar dados. - Cada valor pode ser acessado por um índice dentro da tupla.
### TUPLAS - Semelhante a Listas, exceto que Tuplas são imutáveis: uma vez definidas, elas permanecem iguais por toda execução do programa.

Tuplas

- Tuplas, assim como as listas, são posições da memória: - Identificadas por um único nome - individualizadas por índices - com conteúdo do mesmo tipo ou tipos variados

### Porque usar Tuplas? - Tuplas são mais rápidas que listas - Tuplas são mais seguras que listas - Tuplas são usadas em chaves de dicionários - Tuplas são usadas na formatação de strings

Criação de Tuplas

Para criar uma tupla, utilizam-se parênteses ()
```python #Tupla vazia: T1 = () # Tupla homogênea: T2 = (1,2,3,4,5) # Tupla heterogênea: T3 = ('fisica', 'quimica', 1997, 2000) # Tupla de um elemento: T4 = (50,) #Tupla a partir de uma lista T5 = tuple([3,2,7,4,1]) ```

T[inicio:fim]

retorna os elementos do início ao fim de T. Isto chama-se fatiamento de tuplas.
```python T = (3,2,7,4,1) seq1 = T[1:4] seq2 = T[:3] seq3 = T[:] seq4 = T[:-1] seq5 = T[::2] #T[inicio:fim:n] ```

Não é possivel MODIFICAR elemento de uma tupla

```python T = (3,2,7,4,1) T[4] = 8 #erro!!! ```

TUPLAS

É possível criar tuplas a partir de tuplas.
```python T1 = (3,2,7,4,1) T2 = ('a', 'b', 'c') T3 = T1 + T2 # concatenação de tuplas print (T3) ```

Não é possivel REMOVER elemento de uma tupla

```python T = (3,2,7,4,1) del T[2] #erro!!! ```

TUPLAS

É possível criar uma nova tupla sem um determinado item.
```python T1 = (3,2,7,4,1) T2 = T1[:2] + T1[3:] print (T2) ```

len( )

retorna o número de itens da tupla T
```python T = (3,2,7,4,1) n = len(T) ```

Pertence a Tupla

Verifica se um item pertence ou não a tupla
```python T = (3,2,7,4,1) print(3 in T) ```

Máximo e Mínimo em uma tupla

```python T = (3,2,7,4,1) print(max(T)) print(min(T)) ```

Empacotamento / Desempacotamento

Empacotar significa agrupar valores.
É utilizado para permitir múltiplas atribuições de valores simultaneamente.

Empacotamento

```python T = ('Carlos', 'BICT', '2018.2') ```

Desempacotamento

```python (nome, curso, ingresso) = T ```

Empacotamento / Desempacotamento

Como trocar o valor de duas variáveis a e b?
```python a = 10 b = 5 temp = a a = b b = temp ```

Empacotamento / Desempacotamento

E como fazer a troca utilizando empacotamento?
```python a = 10 b = 5 (a,b) = (b,a) ```

Retorno de funções

Funções podem retornar até um valor apenas
```python def circunferencia(r): c = 2 * math.pi * r return c ```

Retorno de funções

Funções podem retornar até um valor apenas
E se quisermos retornar mais de um valor?
```python def circunferencia_e_area(r): c = 2 * math.pi * r a = math.pi * r * r return (c,a) ```
