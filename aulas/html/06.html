Dicionários (Criação)

Para criar um dicionário, utilizam-se chaves { }
```python # Dicionário vazio D1 = {} # Dicionário homogênio D2 = {'MA' : 'Maranhão', 'PA' : 'Pará', … } # Dicionário heterogênio D3 = {'um':1,'dois':2,'tres':3} D4 = {1:'oi',2:'ola',3:'tchau'} D5 = {(1,2):'tupla!',True:'bool!','t':'string!'} D6 = {[3,4]:'lista!'} #Erro! ```

Dicionários (Adição)

Por adição de Elementos
```python D = {} D['um'] = 'one' D['dois'] = 'two' D['tres'] = 'three' print(D) ```

Dicionários (Criação)

Obs: As chaves devem ser únicas.
```python D = {'Nome': 'Ana', 'Idade': 27, 'Nome': 'Maria'} print(D) ```

Dicionários (Leitura)

D[chave] retorna o valor correspondente a chave.
```python Registro = {'Nome':'Ana', 'CPF':'123.456.789-00', 'Idade':27} print(Registro['Nome']) print(Registro['CPF']) print(Registro['Idade']) ```
OBS: Se a chave não estiver presente, um erro é gerado.
```python print(Registro['Endereco']) #ERRO!! ```

Iteração

Utilizando o for-in: iterar pelos valores
```python D = {'Brasil':'Real','EUA':'Dolar', 'Japao':'Iene'} for m in D.values(): print('A moeda eh: ' + m) ```

Iteração

Utilizando o for-in: iterar pelas chaves
```python D = {'Brasil':'Real','EUA':'Dolar', 'Japao':'Iene'} for p in D.keys(): print('O pais eh: ' + p) ```

Iteração

Utilizando o for-in: iterar pelos elementos
```python D = {'Brasil':'Real','EUA':'Dolar', 'Japao':'Iene'} for (p,m) in D.items(): print('O pais ' + p + ' tem a moeda ' + m) ```

Dicionário (Atualização)

Modifica-se o valor de um elemento por sua chave.
```python D = {'Codigo':'1234567','Prova':7.5} D['Prova'] = 9.0 #Atualiza elemento existente! D['Prova'] += 1.0 D['Trabalho'] = 5.5 #Inclui a nova chave ```

Dicionário (Deleção)

Pode-se remover um elemento específico, todos os itens ou o dicionário inteiro
```python del D['Prova'] # Remove um elemento específico D.clear() # Remove todas os elementos e del D #Remove o dicionário ```

Métodos Auxiliares

len() - Fornece o tamanho do dicionário
```python D = {'Brasil':'Real','EUA':'Dolar', 'Japao':'Iene'} print(len(D)) # Saída: 3 ```

Métodos Auxiliares

keys() - Fornece uma lista contendo todas as chaves do dicionário
```python D = {'Brasil':'Real','EUA':'Dolar', 'Japao':'Iene'} print(D.keys()) # Saída: dict_keys(['Brasil', 'EUA', 'Japao']) ```

Métodos Auxiliares

values() - Fornece uma lista contendo todas os valores do dicionário
```python D = {'Brasil':'Real','EUA':'Dolar', 'Japao':'Iene'} print(D.values()) # Saída: dict_values(['Real', 'Dolar', 'Iene']) ```

Métodos Auxiliares

items() - Fornece uma lista de tupla contendo todas os chaves e valores do dicionário
```python D = {'Brasil':'Real','EUA':'Dolar', 'Japao':'Iene'} print(D.items()) # Saída: dict_items([('Brasil', 'Real'), ('EUA', 'Dolar'), ('Japao', 'Iene')]) ```

Métodos Auxiliares

update() - Adiciona um dicionário a outro do dicionário
```python D1 = {'Brasil':'Real','EUA':'Dolar', 'Japao':'Iene'} D2 = {'Argentina':'Peso','Reino Unido':'Libra'} D1.update(D2) print(D1) ```
```python # Saída: {'Brasil': 'Real', 'EUA': 'Dolar', 'Japao': 'Iene','Argentina': 'Peso', 'Reino Unido': 'Libra'} ```
